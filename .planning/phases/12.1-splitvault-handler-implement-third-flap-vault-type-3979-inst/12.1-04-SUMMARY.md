# Phase 12.1 Plan 04: splitVaultHandler Implementation

Created the third Flap vault handler that reads `userBalances(creator) → (accumulated, claimed)` and returns the difference in BNB wei.

## What Was Built

`lib/platforms/flap-vaults/split-vault.ts` — implements `FlapVaultHandler` interface:

```typescript
export const splitVaultHandler: FlapVaultHandler = {
  kind: 'split-vault',
  async readClaimable(vault, user, _signal) {
    try {
      const [accumulated, claimed] = await bscClient.readContract({
        address: vault,
        abi: SPLITVAULT_USERBALANCES_ABI,
        functionName: 'userBalances',
        args: [user],
      });
      if (claimed > accumulated) return 0n; // defensive invariant
      return accumulated - claimed;
    } catch (err) {
      log.warn('splitVault.readClaimable_failed', { vault: vault.slice(0, 10), error });
      return 0n;
    }
  },
};
```
